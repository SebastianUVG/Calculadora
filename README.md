# Calculadora React

Este es un proyecto de calculadora desarrollado con React y Vite.

## Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/SebastianUVG/Calculadora
cd Calculadora
```
2. Instala todo los paquetes necesarios
```bash
npm install
```

3. Pruebalo en localhost
```bash
npm run dev
```
4. Para producción
```bash
npm run build
```
